Step 1: Setup Virtual Environment 
`virtualenv venv`

Step 2: Activate Virtual Environment
`source venv/bin/activate`

Step 3: Install Requirements Packages
`pip install -r requirements.txt`

Step 4: Run Flask Server
`python app.py`
